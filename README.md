
# Orion Decor Haven

A modern, multi-page responsive website for a home decor brand called Orion. The site features a clean, minimalist UI with soft whites, light grays, and pastel accent colors.

## Project info

**URL**: https://lovable.dev/projects/a2f66aa7-223c-4f87-912f-df58eb9db3a9

## Features

- **Home Page** – Hero banner with featured products, navigation bar, call-to-action buttons, and trending products
- **Products Page** – Grid layout of items with filtering and sorting options
- **Categories Page** – Card-style layout for different home decor categories
- **Cart Page** – User-friendly cart interface with product thumbnails and checkout options
- **Profile/Login Page** – Stylish login/signup forms with modern design

## Design Elements

- Clean, minimalist UI
- Soft whites, light grays, and accent colors in pastel tones (soft green, peach, lavender)
- Rounded corners, subtle shadows, and smooth animations for hover and transitions
- Inter font family
- Mobile-first responsive design
- Sticky navigation bar with icons

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- shadcn/ui components
- React Router
- Lucide React icons

## How to Run Locally

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server with auto-reloading and an instant preview
npm run dev
```

## Deployment

Simply open [Lovable](https://lovable.dev/projects/a2f66aa7-223c-4f87-912f-df58eb9db3a9) and click on Share -> Publish.
